## Development

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```
git clone https://github.com/laneshetron/scratchpad.git
cd scratchpad
```

2. Install dependencies:
```
npm install
```

3. Run in development mode:
```
npm run build && npm run start
```
